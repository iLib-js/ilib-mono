toArray() → {Array.<Node>}

Description:
  • Flatten the current node and all of its descendents into an array and return it. When a node has children, it is flattened into two nodes: a start node, followed by nodes for all its children, and an end node. The start and end nodes are marked by a "use" property set to "start" and "end".

(static) fromArray(array) → {Node}

Description:
  • Recreate a full tree again from a flattened array of Node instances. If the instances are well-formed (that is, all start nodes are matched with end nodes with strict nesting), then the tree is valid. If the array is not well-formed, then the shape of the resulting tree will probably not be valid and the results of this static method are not defined.
