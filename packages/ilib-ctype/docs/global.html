(constant) propertyAliases

Description:
  • Map from general-category aliases (lower-cased) to the short property codes used as keys inside the ctype_* data tables. Callers may pass either the short code ("Mn") or the long Unicode name ("Nonspacing_Mark"); both resolve to the same internal lookup.
