TMX

TMX

A class that represents an tmx 1.4b file. See https://www.gala-global.org/tmx-14b for details on the file format.

Constructor

new TMX(options)

Description:
  • Construct a new TMX file. The options may be undefined, which represents a new, clean Tmx instance. The options object may also be an object with the following properties:
    • path - the path to the tmx file on disk
    • sourceLocale - specify the default source locale if a resource doesn't have a locale itself. Default is "en".
    • version - The version of tmx that will be produced by this instance. Default is "1.4".
    • properties - an object containing general string properties that will appear in the header of the tmx file. Typical properties are:
      • creationtool - the full name of the tool that created this tmx file. Default: "loctool"
      • creationtoolversion - the version of the tool that created this tmx file. Default: the version of this loctool
      • originalFormat - the format of the data before it was transformed into tmx. That can be any string.
      • datatype - the data type of the strings that these translations originally came from
    • segmentation - How the strings should be segmented. Choices are "paragraph" and "sentence." Default is "paragraph". The tmx settings of "block" and "phrase" are not yet supported.

addResource(res)

Description:
  • Add a resource to this tmx file. If a resource with the same file, locale, context, and key already exists in this tmx file, what happens to it is determined by the allowDups option. If this is false, the existing resource will be replaced, and if it is true, this new resource will be added as an instance of the existing resource.

diff(other) → {TMX}

Description:
  • Compare the other TMX instance with the current one and return a new TMX instance with the difference. This method only handles the difference from the current tmx to the other tmx. That is, it handles changes and additions in the other tmx, but not deletions from the current tmx. For translation memories, the result should be a superset of translations that are possible for the source text so deletions are not necessary.
