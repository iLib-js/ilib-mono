getBySource(source, context) → {Resource|undefined}

Description:
  • Get a resource by its source string and context. The source string must be written in the language and script of the source locale. For array types, the source string must be one of the values in the string array. For plural types, it must be one of the values of the quantities.

    If the context is undefined, this method will find the base generic resource with no context.
